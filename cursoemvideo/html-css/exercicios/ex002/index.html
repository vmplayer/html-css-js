Parágrafos e quebras de linhas

Veja o código para ver como fazer.

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags <p> e </p>.

e

Se precisar quebrar um texto em qualquer lugar específico
como esse você pode usar a tag <br>.

Vamos adicionar algumns símbolos especiais:

Marca Registrada: ®
Copyright: ©
Trade Mark: ™
Moedas: € £ ¥ ¢
Símbolos matemáticos: Δ δ ℜ π
Setas: ↑ ↓ ← →.

Vamos adicionar alguns emojis!

Sempre que for adicionar um emoji, procure pelo "Codepoint" do emoji e assim, adicione-o ao código usando os padrões de valores hexadecimais.
Adicione o código usando "&xcódigo_após_o_traço" (&#x CODEwithoutSpace;)
